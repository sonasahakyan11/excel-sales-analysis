# Excel Sales Analysis Project

This project demonstrates Excel skills for analyzing sales data.

## Skills Used
- Excel Tables
- IF function
- Filtering & Sorting
- SUBTOTAL (dynamic calculations)

## Description
Created an interactive sales report that updates total and average sales based on filters.

## Files
- sales-analysis-v1.xlsx
